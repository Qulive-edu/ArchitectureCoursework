# ParkRent — Сервис бронирования парковочных мест

## Описание проекта
Полнофункциональное клиент-серверное веб-приложение, разработанное в рамках курсовой работы по дисциплине «Разработка клиент-серверных приложений» (РТУ МИРЭА, Институт информационных технологий, кафедра ИиППО). Система обеспечивает поиск, просмотр и резервирование парковочных площадок с поддержкой транзакционной обработки запросов, безопасной аутентификации пользователей и автоматизированного развёртывания в контейнеризированной среде.

| **CI/CD**        | GitHub Actions, GitHub Container Registry, автоматические миграции, rollback |

## Архитектура системы
Проект реализует трёхуровневую клиент-серверную архитектуру с чётким разделением ответственности:
1. **Presentation Tier** — одностраничное приложение (SPA) на React, обеспечивающее взаимодействие с пользователем, валидацию форм и асинхронный обмен данными.
2. **Application Tier** — серверная часть на Go, реализующая маршрутизацию, бизнес-логику, аутентификацию и управление транзакциями. Архитектура следует принципам Clean Architecture (Controller → UseCase → Repository).
3. **Data Tier** — PostgreSQL для хранения структурных данных и обеспечения ссылочной целостности; Redis для быстрых операций чтения/записи и управления сессиями.

Взаимодействие между уровнями осуществляется через REST API по протоколу HTTP. Входящие запросы обрабатываются обратным прокси Nginx, который маршрутизирует вызовы `/api/*` на пул бэкенд-контейнеров и отдаёт статические файлы фронтенда.
